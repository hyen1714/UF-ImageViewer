# UF-ImageViewer
WPF 기반 2D 이미지 뷰어

## 🖥️ 프로젝트 소개
2D 이미지 관련 영상처리의 편리함을 위하여 만들어 보았습니다.

## ⚙️ 개발 환경
- `WPF`
- `.NET 7.0`
- **IDE** : Visual Studio 2022

## 📌 주요 기능
- 확대 및 축소
- 확대 후 이미지 이동
- 드래그 앤 드롭
